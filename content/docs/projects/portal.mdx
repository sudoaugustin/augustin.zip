## Notes
- Portal is safe to use and doesn't collect any user data.
- Portal overrides some network requests to embed a website in a safer way using [declarativeNetRequest](https://developer.chrome.com/docs/extensions/reference/api/declarativeNetRequest#:~:text=declarativeNetRequest%20API%20is%20used%20to,content%2C%20thus%20providing%20more%20privacy.).
- Portal ensures the network requests are overridden only in the homepage and doesn't effect your other webpages.
- Although Portal overrides network requests, some websites use javascript to detect if it's embedded. Portal may not work well with those websites.
