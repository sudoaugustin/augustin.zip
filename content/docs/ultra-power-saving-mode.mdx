# Ultra Power Saving Mode for Myanmar

![Ultra Power Saving Mode](/images/writings/ultra-battery-banner.png)

မြန်မာနိုင်ငံတွင် 7.7 မက်ဂနီကျုဒ်ငလျင်ကြီးဖြစ်ပွားပြီးနောက် လျှပ်စစ်ဓာတ်အားရရှိမှုအခက်အခဲများနှင့် ရင်ဆိုင်နေရပါတယ်။ ဒီလိုအချိန်မာ ဘက်ထရီအားကို ထိရောက်စွာသုံးစွဲပြီး မိသားစုနှင့် ချစ်သူများနှင့် ဆက်သွယ်ရန်၊ အရေးပေါ်ဝန်ဆောင်မှုများကို ရယူရန်အတွက် အရေးကြီးပါသည်။ ဒီအချိန်မှာ Ultra Power Saving Mode (UPSM) ဟာ သင့်ဘဝကို ကယ်တင်ပေးနိုင်တဲ့ အရေးပါတဲ့ feature တစ်ခုဖြစ်ပါတယ်။

## Ultra Power Saving Mode ဆိုတာဘာလဲ?

Ultra Power Saving Mode ဟာ ဖုန်းတွေမှာ ပါဝင်တဲ့ feature တစ်ခုဖြစ်ပြီး ဘက်ထရီအားကို သိသိသာသာ လျှော့ချပေးနိုင်ပါတယ်။ ဒီ mode ကို အသုံးပြုတဲ့အခါ ဖုန်းမှာ အသုံးလိုအပ်တဲ့ app တွေ၊ ဝန်ဆောင်မှုတွေကိုသာ အသုံးပြုနိုင်ပြီး အခြား feature တွေကို ပိတ်ထားပါတယ်။

ဥပမာအားဖြင့် Samsung ဖုန်းတွင် Ultra Power Saving Mode ကို အသုံးပြုပါက ဘက်ထရီ 10% သာကျန်ရှိနေတဲ့အချိန်မှာတောင် 24 နာရီထိ standby လုပ်နိုင်ပါတယ်။

## Ultra Power Saving Mode ဘယ်လိုအလုပ်လုပ်သလဲ?
Ultra Power Saving Mode ကို ဖွင့်လိုက်တဲ့အခါ ဖုန်းမှာ အောက်ပါအတိုင်း ပြုပြင်လိမ့်မည်-

- App အသုံးပြုမှုကို ကန့်သတ်ခြင်း: Phone, Messages, Internet စသော အရေးကြီး app များကိုသာ အသုံးပြုနိုင်ပါသည်။

- Screen Brightness လျှော့ချခြင်း: Display ကို အနက်နှင့် အဖြူရောင် (grayscale) mode သို့ ပြောင်းလဲပေးသည်။

- မလိုအပ်သော Function များ ပိတ်ထားခြင်း: Wi-Fi, Bluetooth, GPS, Background App Activity စသည်တို့ကို ပိတ်ထားသည်။

- CPU Performance ကို ထိရောက်စွာ လျှော့ချခြင်း: Processor speed ကို လိုအပ်သလောက်သာ အသုံးပြုသည်။
